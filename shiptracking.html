위성 AIS 데이터 서비스

인공위성에서 수집한 선박위치발신기AIS(Automatic Identification System) 데이터를 고객에게 API형태로 제공해 주는 서비스. 실시간 데이터 또는 과거데이터를 기간, 영역, 척수별로 JSON 형식 제공

선박 위치 모니터링 웹서비스

인공위성에서 수집한 선박위치 발신기AIS(Automatic Identification system)의 정보를 활용하여 전 세계 선박의 현재 위치를 지도 위에 표시해 주는 고객 맞춤형 지도 서비스

(웹 서비스 기능)

관심 선박 그룹 설정

인공위성에서 수집한 선박위치 발신기 AIS의 정보를 활용하여 전 세계 선박의 현재 위치를 지도위에 표시

세계 항구 표시

전 세계의 항구의 이름과 좌표를 표시

주요 영역 표시

EEZ(배타적경제수역),ECA(환경규제지역),해적 출몰지역 등을 표시

출,도착 알림

지도에 영역을 설정한 후 영역을 벗어나거나 들어올 때를 자동으로 탐지하여 선박의 출,도착을 알려줌

기상정보

해상의 날씨 정보를 지도에 표시

미래항로 도착시각 예측

선박의 미래 항로를 예측하여 지도에 표시 해주고 도착시간을 예측

선박 검색

IMO,MMSI,선박명으로 선박검색
